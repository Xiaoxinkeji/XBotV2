# 基础使用指南

本文档将介绍XBotV2的基本使用方法，包括登录、消息发送、插件管理等核心功能。

## 首次启动和登录

1. 启动XBotV2后，在浏览器中访问管理界面：`http://localhost:8080`（或您配置的其他地址）。
2. 默认情况下，您会看到一个登录页面，点击"微信登录"按钮。
3. 系统会显示一个二维码，使用您的微信扫描此二维码。
4. 在微信中确认登录后，Web界面会自动刷新，显示主控制面板。

![登录流程](../assets/images/login_flow.png)

## 管理界面概览

登录成功后，您将看到XBotV2的主界面，包含以下主要区域：

1. **顶部导航栏**：提供对主要功能的快速访问
2. **左侧边栏**：包含所有功能模块的导航菜单
3. **中央内容区**：显示当前选定功能的详细信息和控制选项
4. **状态栏**：显示系统状态、在线状态和重要通知

主要功能区域包括：

- **仪表盘**：概览系统状态和关键数据
- **消息记录**：查看历史消息记录
- **联系人管理**：管理微信联系人和群组
- **插件管理**：启用、禁用和配置插件
- **插件市场**：浏览、安装和更新插件
- **系统设置**：配置系统参数和选项

## 与联系人互动

### 查看联系人列表

1. 在左侧导航栏中点击"联系人"。
2. 系统将显示您微信账号中的所有联系人，包括：
   - 个人联系人
   - 群聊
   - 公众号

您可以使用搜索框根据名称或微信ID快速查找特定联系人。

### 发送消息

1. 在联系人列表中找到并选择目标联系人。
2. 点击联系人卡片或"发送消息"按钮。
3. 在打开的聊天窗口中，输入您想发送的消息。
4. 点击"发送"按钮或按Enter键发送消息。

XBotV2支持发送多种类型的消息：

- **文本消息**：直接输入文本并发送
- **图片消息**：点击聊天窗口中的图片图标，选择或拖拽图片
- **文件消息**：点击文件图标，选择要发送的文件
- **语音消息**：点击麦克风图标，录制语音消息

### 查看聊天记录

1. 在联系人列表中选择一个联系人。
2. 系统会自动加载与该联系人的最近聊天记录。
3. 滚动查看历史消息，或使用搜索功能查找特定消息。

## 插件管理

XBotV2的功能可以通过插件进行扩展。以下是如何管理插件的基本步骤：

### 查看已安装插件

1. 在左侧导航栏中点击"插件管理"。
2. 系统将显示所有已安装的插件列表，包括：
   - 插件名称和描述
   - 当前状态（启用/禁用）
   - 版本信息
   - 作者信息

### 启用或禁用插件

1. 在插件列表中找到目标插件。
2. 点击插件卡片上的开关按钮即可启用或禁用该插件。
3. 系统会实时更新插件状态，无需重启。

### 配置插件

1. 在插件列表中找到需要配置的插件。
2. 点击插件卡片上的"设置"按钮。
3. 在打开的配置面板中修改插件参数。
4. 点击"保存"按钮应用更改。

每个插件的配置选项会有所不同，请参考具体插件的文档。

## 插件市场使用

XBotV2提供了插件市场，让您可以轻松发现、安装和更新插件。

### 浏览插件市场

1. 在左侧导航栏中点击"插件市场"。
2. 系统将显示可用插件的列表，包括官方插件和社区插件。
3. 您可以通过分类、标签或搜索来过滤插件。

### 安装新插件

1. 在插件市场中找到您想安装的插件。
2. 点击插件卡片上的"安装"按钮。
3. 系统会自动下载并安装插件。
4. 安装完成后，您可以在"插件管理"页面中找到并启用它。

### 更新插件

1. 在插件市场中，已安装的插件如有更新会显示"更新"按钮。
2. 点击"更新"按钮即可升级到最新版本。
3. 也可以在"插件管理"页面中点击"检查更新"按钮一次性检查所有插件的更新状态。

## 系统设置

### 基本设置

1. 在左侧导航栏中点击"系统设置"。
2. 在"基本设置"标签页中，您可以配置：
   - 机器人名称
   - 管理员账号
   - 自动回复设置
   - 工作时间设置

### 网络设置

在"网络设置"标签页中，您可以配置：
   - API服务器地址和端口
   - Web界面访问设置
   - 代理设置（如需要）

### 高级设置

在"高级设置"标签页中，您可以配置：
   - 日志级别和保存路径
   - 数据库设置
   - 性能调优选项

修改任何设置后，点击"保存"按钮应用更改。某些更改可能需要重启系统才能生效。

## 常用命令

XBotV2支持通过命令行方式控制机器人，以下是一些常用命令：

### 基本命令

- `/help`：显示帮助信息
- `/status`：查看系统状态
- `/restart`：重启机器人
- `/stop`：停止机器人

### 插件相关命令

- `/plugins`：列出所有插件
- `/enable [插件名]`：启用指定插件
- `/disable [插件名]`：禁用指定插件
- `/reload [插件名]`：重新加载指定插件

### 消息相关命令

- `/send [联系人] [消息]`：向指定联系人发送消息
- `/broadcast [消息]`：向所有联系人广播消息（谨慎使用）

## 用户权限管理

XBotV2具有基本的权限管理功能，可以控制不同用户对机器人的使用权限。

### 权限级别

系统定义了以下权限级别：
1. **管理员**：拥有完全控制权
2. **高级用户**：可以使用大部分功能，但不能修改系统设置
3. **普通用户**：只能使用基本功能
4. **受限用户**：功能使用受到严格限制

### 设置用户权限

1. 在左侧导航栏中点击"用户管理"。
2. 找到并选择目标用户。
3. 点击"编辑权限"按钮。
4. 在弹出的对话框中设置用户权限级别。
5. 点击"保存"按钮应用更改。

## 高级功能

### 定时任务

1. 在左侧导航栏中点击"定时任务"。
2. 点击"新建任务"按钮。
3. 配置任务参数：
   - 任务名称
   - 执行时间（可设置为单次或重复执行）
   - 执行动作（发送消息、执行命令等）
   - 目标联系人（如适用）
4. 点击"保存"按钮创建任务。

### 自动回复规则

1. 在左侧导航栏中点击"自动回复"。
2. 点击"新建规则"按钮。
3. 配置规则参数：
   - 触发关键词或正则表达式
   - 回复内容
   - 适用范围（所有人、特定联系人或群组）
   - 启用状态
4. 点击"保存"按钮创建规则。

### 数据导出与备份

1. 在左侧导航栏中点击"数据管理"。
2. 在"备份与恢复"标签页中：
   - 点击"创建备份"按钮生成系统数据备份
   - 点击"导出数据"按钮导出特定数据（如聊天记录）
   - 点击"导入备份"按钮从备份文件恢复系统

## 常见问题解决

### 登录问题

如果无法登录或微信频繁掉线：
1. 确保您的网络连接稳定。
2. 检查微信账号是否在其他设备登录。
3. 确认XBotV2版本是否为最新。

### 消息发送失败

如果消息无法发送：
1. 检查与目标联系人的好友关系是否正常。
2. 确认系统是否处于在线状态。
3. 查看日志文件了解详细错误信息。

### 插件无法工作

如果插件无法正常工作：
1. 在插件管理中禁用并重新启用该插件。
2. 检查插件配置是否正确。
3. 查看系统日志了解详细错误信息。
4. 尝试更新或重新安装该插件。
